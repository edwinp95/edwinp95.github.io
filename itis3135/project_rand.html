Page Design

  1. Home
    • Purpose: Brief intro of the purpose of the site.
    • Users: Anyone interested in art.
    • Content: Navigation Bar. Artwork slideshow.
    • Data to be Entered: N/A
    • Buttons/Links: Navigation Bar.
    • Actions: Nav. Bar guides user through site with links to other pages.
  2. About
    • Purpose: Background of my client. Any other personal information they wish to share.
    • Users: Anyone interested in getting to know my client.
    • Content: Photo of client. Paragraph of personal information.
    • Data to be Entered: N/A
    • Buttons/Links: Navigation bar.
    • Actions: *
  3. Gallery
    • Purpose: Present my clients artwork collection.
    • Users: Potential customers. Anyone interested in artwork.
    • Content: A complete collection of clients artwork. For sale or not for sale.
    • Data to be Entered: N/A
    • Buttons/Links: Navigation Bar.
    • Actions: *
  4. Contact
    • Purpose: Allow anyone to contact my client given their provided socials.
    • Users: Anyone interested in furthering their interests with my client.
    • Content: All socials. Form for any desired commission work.
    • Data to be Entered: Name. Email. Short paragraph.
    • Buttons/Links: Navigation Bar. Submit button.
    • Actions: Submit sends the request to my client.
  5. Print Shop
    • Purpose: Allow anyone to purchase prints from my client.
    • Users: Anyone interested in buying my clients work.
    • Content: IMages of available prints.
    • Data to be Entered: Which print someone wants to purchase.
    • Buttons/Links: Purchase button.
    • Actions: Purchase button redirects user to buy my clients art.

Dynamic Functionality

  • Buttons that link to other pages.
  • Clickable media gallery.
  • Art slideshow.
  • Form submission on contact and print shop page.
