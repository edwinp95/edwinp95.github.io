Website Evaluations

Yale Art

This site... is a page where all members of the school of art have the ability to edit the page and upload whatever they choose.

  • CONTRAST: This page fails to have clear contrast. There are many colors going on.
  • REPETITION: This page absolutely fails in the area of repetition. Every element is unique and there seems to be no pattern in any of the design choices.
  • ALIGNMENT: ALl the elements on this page are not TOO badly aligned but the quick links seems in an odd location.
  • PROXIMITY: The main elements are in close proximity but alot of information needs to be scrolled down to.

Youtube

This site... is one of the most popular sites on the internet, if not the most popular. It is a platform for whoever to share videos.

  • CONTRAST: This page has a clear contrast and highlights important elements.
  • REPETITION: This page does a great job repeating element choices for a clear pattern such as the thumbnails for the videos and such.
  • ALIGNMENT: Everything on this page is aligned perfectly.
  • PROXIMITY: All the main information a user would ever need is all nearby in the main view next to each other.
